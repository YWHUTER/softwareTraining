# 🤖 校园新闻系统 AI 功能使用指南

## 📋 功能概览

您的校园新闻系统现已集成多项先进的AI功能，让项目更加智能化和现代化！

## ✨ 已实现的AI功能

### 1. 🔍 AI智能摘要生成
**路径**: `/ai-summary`

**功能特点**:
- 支持4种摘要长度（50字、100字、200字、300字）
- 3种写作风格（专业、通俗、学术）
- 自动提取关键要点
- 智能生成标题建议
- 关键词提取与分析
- 摘要质量评分

**使用场景**:
- 快速理解长文章内容
- 为文章生成精炼摘要
- 获取文章核心观点
- 辅助编辑工作

### 2. 💭 AI情感分析
**路径**: `/ai-sentiment`

**功能特点**:
- 总体情感评分（0-100分）
- 情感倾向分析（积极/中性/消极）
- 6种情绪细分（喜悦、信任、期待、悲伤、愤怒、恐惧）
- 关键词情感标注
- 智能改进建议
- 情绪雷达图可视化

**使用场景**:
- 分析新闻报道的情感倾向
- 评估评论的情绪色彩
- 优化文章的情感表达
- 监控舆情动向

### 3. 💬 AI智能对话（原有功能）
**路径**: `/ai-assistant`

**功能特点**:
- 智能问答
- 新闻搜索
- 热门查询
- 数据统计

## 🚀 快速开始

### 前端启动
```bash
cd frontend
npm install
npm run dev
```

### 后端启动
```bash
cd backend
mvn clean install
mvn spring-boot:run
```

## 🎯 API接口文档

### 摘要生成
```javascript
POST /api/ai/enhanced/summary
{
  "content": "文章内容",
  "length": 200,
  "style": "professional"
}
```

### 情感分析
```javascript
POST /api/ai/enhanced/sentiment
{
  "text": "待分析文本"
}
```

### 标题生成
```javascript
POST /api/ai/enhanced/titles
{
  "content": "文章内容",
  "count": 3
}
```

### 质量评估
```javascript
POST /api/ai/enhanced/quality
{
  "content": "文章内容"
}
```
