## 🎨 设计系统

### 颜色规范

```css
/* 主色调 */
--primary-blue: #2196f3;
--primary-dark: #1976d2;

/* 渐变 */
--gradient-primary: linear-gradient(135deg, #2196f3 0%, #1976d2 100%);
--gradient-purple: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 中性色 */
--text-primary: #2c3e50;
--text-regular: #606266;
--text-secondary: #909399;
--border-color: #e4e7ed;
--bg-gray: #f5f7fa;
```

### 圆角规范

```css
--radius-sm: 6px;   /* 标签、小按钮 */
--radius-md: 8px;   /* 输入框、一般按钮 */
--radius-lg: 12px;  /* 卡片 */
--radius-xl: 16px;  /* 大卡片 */
--radius-round: 24px; /* 圆角按钮 */
```

### 阴影规范

```css
--shadow-sm: 0 2px 8px rgba(0, 0, 0, 0.04);
--shadow-md: 0 2px 12px rgba(0, 0, 0, 0.08);
--shadow-lg: 0 4px 20px rgba(0, 0, 0, 0.12);
--shadow-primary: 0 4px 12px rgba(33, 150, 243, 0.3);
```

### 间距规范

- **小间距**：8px, 12px
- **中等间距**：16px, 20px, 24px
- **大间距**：30px, 40px

## 📱 响应式设计

### 断点
- **手机**：< 768px
- **平板**：768px - 1024px
- **PC**：> 1024px

### 适配策略

**登录/注册页：**
- 手机：隐藏左侧区域，单列表单
- 平板：保持左右布局

**主布局：**
- 手机：隐藏菜单文字，仅图标
- 平板：正常显示

**首页：**
- 手机：单列布局，侧边栏移至底部
- 平板：两列布局
- PC：17/7 栅格布局

**文章详情：**
- 手机：单列，操作按钮横向排列
- PC：最大宽度 900px 居中

**发布页面：**
- 手机：单列表单
- PC：两列表单
