## 🔐 安全特性

| 安全措施 | 实现方式 |
|---------|---------|
| 密码加密 | BCrypt加密存储 |
| 身份认证 | JWT Token认证 |
| 权限控制 | 基于角色的访问控制(RBAC) |
| SQL注入防护 | MyBatis参数化查询 |
| XSS防护 | 前端内容转义 |
| CORS配置 | Spring Security跨域配置 |

## 🚀 性能优化

| 优化项 | 实现方式 | 效果 |
|--------|---------|------|
| 浏览量缓存 | Redis缓存 | 减少数据库压力 |
| 分页查询 | MyBatis-Plus分页 | 避免大数据量查询 |
| 数据库索引 | 关键字段索引 | 提升查询速度 |
| 前端路由懒加载 | Vue Router动态导入 | 减少首屏加载时间 |
| API响应缓存 | 浏览量每10次更新一次 | 减少数据库写入 |
